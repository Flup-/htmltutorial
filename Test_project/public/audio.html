Audio Tag

The audio tag is utilized to add audio to a page. It is set up like so:
<audio controls>
<source src="/Directory/filename.extension" type="filetype/extensiontype">>
</audio>
All of this looks complicated so let's break it down using the example at the top of the page. First, however, we must know the 3 audio files types:

  1. mp3 = audio/mpeg
  2. ogg = audio/ogg
  3. wav = audio/wav

Now we can use the example above to show each aspect of the line

  • audio controls - lets you control the audio
  • source/src="/audio/KH2.mp3 - The directory the audio is sourcing from
  • type="audio/mpeg" - this is the audio type. It defines that this file is audio and an mpeg file.

This is the most basic way to host an audio file on your website. It is also good to note that you may use anchors to link this kind of file to your site. Put each aspect into the code and you should get a working audio file! Try to put your own file on your page!
