Buttons in action:

This is actually quite simple and is effectively done in one line:
<a href="index.html"><button>Home</button></a>
This seems super complicated but its not so bad. Let's Break it down:

To begin, it is place in an anchor tag which is simply making sure that it has a hyperlink attached to the button. href="index.html" refers to the html page that you started on. Then it is nothing but the basic button tag wrapped up in that anchor.
