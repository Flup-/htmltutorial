How to insert images.

We are going to learn how to put this amazing picture into your HTML.

The image tag, <img> has two required attributes: src and alt. src gives the source to the html in order to find the file and alt provides additional information if for some reason the file cannot load.

The code is fairly simple and looks like this:
<img src="/directory/filename.extension" alt="further info">
You can also set the width and height with width="X" and height="Y". As you can see, the image tag does not end with <img>.
The code for the above image looks like this:

<img src="/images/smiles.jpg" alt="Smiley face">
The tag is made and the first thing that it does is source the file in my HTML project's images folder. Then it adds the alt and boom its there. Simple as that.

Images can also be sourced from websites through anchors that will redirect pages images to yours.
