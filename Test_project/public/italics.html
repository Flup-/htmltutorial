This is the tutorial for COLOR, ITALICS and EMPHASIS.

Italics and Bold

Using italics and bold is very simple and easy to do. To begin you want to know the tags <i> and <b>. These simple stand for Italics and Bold. in your paragraph, header, etc you will put the tags around the text that you want to change and voila, you have Italics and Bold

Code:

<b>TEXT</b> and <i>TEXT</i>

Becomes: TEXT and TEXT

The Font Tag

The font tag deals with a lot of things including:

Color
<font color="COOL COLOR HERE"> TEXT </font>
Font Size
<font size="1,2,3,4,5,or 6"> TEXT </font>

These are used as indicated and can will show up like so: COOL STUFF HERE

You are now able to use the above tags in order to stylize your texts. If you want to change the font, the face tag is no longer supported and you should go learn some CSS :)
