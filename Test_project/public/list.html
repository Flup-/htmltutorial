How to code a List

Types of lists:

Unordered List
- A simple bullet list
Ordered List
- A simple Number list
Decription List
- A list with descriptions (This is a description list)

Unordered List

This simply makes a bullet list

This is done with:
<ul>
<li>TEXT</li>
</ul>

That will do this:

  • TEXT

Ordered List

This simply makes a bullet list

This is done with:
<ol>
<li>TEXT</li>
</ol>

That will do this:

  1. TEXT

Description List

This simply makes a bullet list

this is done with:
<dl>
<dt>ITEM</dt>
<dd>- DESCRIPTION</dd>
</dl>

That will do this:

ITEM
- DESCRIPTION
