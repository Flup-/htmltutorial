How to implement videos.

We will be using this video as an example to benefit your learning experience

To begin we need to understand the video tag and its attributes,
<video width=X height=Y controls>
<source src="VIDEO.extension" type="video/extensiontype">
</video>

These are the file types:

  • mp4 = video/mpeg
  • webM = video/webm
  • ogg = video/ogg

You may then use these to write your own file code. The one above looks like this:
<video width="320" height="240" controls>
<source src="/video/I CANT EVEN.mp4" type="video/mp4">
</video>

If you follow the above steps. It should be easy enough to put in any compatibled video to your website. It is also important to note you are able to use anchors to link them straight to your page in certaiin situations.
